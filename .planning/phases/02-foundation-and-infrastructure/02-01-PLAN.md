       - `FileType = 'code' | 'text' | 'image'`
       - `ScannedFile = { absolutePath: string; relativePath: string; type: FileType; sizeBytes: number }`
       - `ScanOptions = { useIgnoreFiles: boolean; typeFilter?: FileType }`
       - `ModelBackend = 'webgpu' | 'cpu'`
       - `CollectionName = 'col-768' | 'col-512'`
       - Extension map constant: `EXTENSION_MAP: Record<string, FileType>` covering .ts, .tsx, .js, .jsx, .py, .go, .rs, .java, .c, .cpp, .h, .hpp, .rb, .php, .swift, .kt, .scala, .sh, .bash, .zsh, .css, .scss, .html (code); .md, .mdx, .txt, .rst (text); .jpg, .jpeg, .png, .gif, .webp, .svg (image)

    5. Create `src/config/paths.ts`:
       - `resolveProjectStoragePath(projectDir: string): string` — uses `crypto.hash('sha256', path.resolve(projectDir)).slice(0, 8)` for 8-char hash, returns `~/.ez-search/<basename>-<hash>/`
       - `resolveModelCachePath(): string` — returns `~/.ez-search/models/`
       - Both use `os.homedir()` and `path.join`

       - Stub body: `console.log('index command stub', { targetPath, ...options })`
