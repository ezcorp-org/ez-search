  <action>
    Create `src/services/chunker.ts` following the verified research pattern exactly.

    **Constants:**
    - `CHUNK_SIZE = 500` (tokens per chunk)
    - `OVERLAP = 50` (token overlap between consecutive chunks)

    **Chunk interface:**
    ```
    { text: string, lineStart: number, lineEnd: number, chunkIndex: number, tokenCount: number }
    ```
    - `lineStart` and `lineEnd` are 1-indexed
    - `chunkIndex` is 0-indexed position within the file

    **loadTokenizer():**
    - Import `AutoTokenizer` and `env` from `@huggingface/transformers`
    - Import `PreTrainedTokenizer` type from `@huggingface/transformers`
    - Set `env.cacheDir` to `resolveModelCachePath()` from `../config/paths.js`
    - Set `env.allowRemoteModels = true`
    - Return `AutoTokenizer.from_pretrained('jinaai/jina-embeddings-v2-base-code')`
    - Return type: `Promise<PreTrainedTokenizer>`

    **chunkFile(text: string, tokenizer: PreTrainedTokenizer): Chunk[]:**
    1. Split text into lines with `text.split('\n')`
    2. Build cumulative token count per line:
       - For each line, encode with `tokenizer.encode(lineText, { add_special_tokens: false })`
       - Append newline to each line except the last before encoding
       - The encode return is array-like (has `.length`, is iterable) but not a true Array -- access `.length` directly
       - Accumulate token count in `cumulative[]` array
    3. Encode full text with `tokenizer.encode(text, { add_special_tokens: false })`
       - Use `add_special_tokens: false` because the embedding pipeline adds special tokens at inference time
    4. Single-chunk case: if total tokens <= CHUNK_SIZE, return single chunk spanning all lines
    5. Multi-chunk sliding window:
       - `stride = CHUNK_SIZE - OVERLAP` (450 tokens)
       - For each window starting at `start`, ending at `min(start + CHUNK_SIZE, totalTokens)`:
         - Slice token IDs for the window
         - Decode with `tokenizer.decode(chunkIds, { skip_special_tokens: true })`
         - Map start token index to lineStart via `tokenIndexToLine()`
         - Map `end - 1` token index to lineEnd via `tokenIndexToLine()`
         - Break if end reaches totalTokens
    6. **tokenIndexToLine(tokenIdx, cumulative):** Linear scan of cumulative array -- return `i + 1` (1-indexed) for the first `i` where `tokenIdx < cumulative[i]`. If not found, return `cumulative.length`.

    **Important implementation notes:**
    - `tokenizer.encode()` returns an array-like object, not a plain Array. Use `Array.from()` if you need array methods like `.slice()`, or cast appropriately.
    - Do NOT call `loadTokenizer()` inside `chunkFile()` -- the tokenizer must be loaded once externally and passed in.
    - Export `CHUNK_SIZE` and `OVERLAP` constants so the index command can reference them for documentation/logging.
  </action>
